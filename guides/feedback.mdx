## Overview

Mizen includes a built-in feedback system that allows you to report bugs, request features, and share suggestions directly from the app. Your feedback helps improve the product for everyone.

## Submitting feedback

Feedback is submitted through the Notion integration, which stores your reports in a structured database for review and prioritization.

### What you can report

- **Bug reports** - Issues or errors you encounter while using Mizen
- **Feature requests** - Ideas for new features or improvements
- **General feedback** - Suggestions, questions, or comments

### Information included

When you submit feedback, the following information is automatically collected:

- **Type** - Bug report, feature request, or general feedback
- **Title** - Brief summary of your feedback
- **Message** - Detailed description
- **Device/OS** - Your operating system and device information
- **App version** - The version of Mizen you're using
- **Screenshots** - Optional images to illustrate your feedback (up to 3 images, 10MB each)

<Note>
Feedback submissions require the Notion integration to be configured. If feedback is unavailable, you can still report issues via GitHub.
</Note>

## Supported file types

Screenshot uploads support the following image formats:

- JPEG (.jpg, .jpeg)
- PNG (.png)
- WebP (.webp)
- HEIC (.heic)
- HEIF (.heif)

**Maximum file size**: 10MB per image  
**Maximum screenshots**: 3 images per feedback submission

## Privacy

- Feedback is stored in a private Notion database accessible only to the development team
- No personal information beyond what you provide is collected
- Screenshots are uploaded to Notion's secure storage

## Alternative feedback channels

If the in-app feedback system is unavailable, you can:

- **Open a GitHub issue** at [github.com/parse-n-plate/mizen/issues](https://github.com/parse-n-plate/mizen/issues)
- **Submit a pull request** with improvements or bug fixes

See the [contributing guide](/development/contributing) for more information on contributing to Mizen.
