Mizen includes keyboard shortcuts to help you navigate faster and complete common actions without reaching for your mouse.

## Global Shortcuts

These shortcuts work anywhere in the application.

| Shortcut | Action | Description |
|----------|--------|-------------|
| `⌘K` or `Ctrl+K` | Open Command Palette | Search recipes and access quick actions |
| `⌘L` or `Ctrl+L` | Copy Recipe Link | Copy the current recipe's source URL to clipboard |

## Recipe Tab Navigation

Quickly switch between recipe tabs from any recipe page.

| Shortcut | Action | Description |
|----------|--------|-------------|
| `⌘1` or `Ctrl+1` | Prep Tab | Switch to ingredients view |
| `⌘2` or `Ctrl+2` | Cook Tab | Switch to cooking instructions |
| `⌘3` or `Ctrl+3` | Plate Tab | Switch to plating and photos |

<Note>
These shortcuts only work when you're viewing a recipe page. They won't trigger if you're typing in an input field.
</Note>

## Cook Mode Shortcuts

These shortcuts are available in the Cook tab.

| Shortcut | Action | Description |
|----------|--------|-------------|
| `⌘J` or `Ctrl+J` | Toggle View | Switch between List view and Focus view |
| `⌘L` or `Ctrl+L` | Copy Link | Copy recipe URL to clipboard |

### Focus View Navigation

When in Focus view (single-step card view):

| Action | Method |
|--------|--------|
| Next Step | Click right arrow or swipe left (mobile) |
| Previous Step | Click left arrow or swipe right (mobile) |
| Return to List | Click "List" button or press `⌘J` |

## Mobile Gestures

On mobile devices (touchscreens), use swipe gestures to navigate:

| Gesture | Action |
|---------|--------|
| Swipe left | Move to next tab (Prep → Cook → Plate) |
| Swipe right | Move to previous tab (Plate → Cook → Prep) |

<Tip>
Swipe gestures work on the recipe page and in Focus view. They provide a natural way to navigate without tapping buttons.
</Tip>

## Input Field Shortcuts

These shortcuts work in specific input contexts.

### Search Bar

| Shortcut | Action |
|----------|--------|
| `Enter` | Parse recipe (when URL is entered) |
| `Esc` | Clear search / close dropdown |

### Ingredient Search

| Shortcut | Action |
|----------|--------|
| `Esc` | Clear search filter |

### Rename Dialog

| Shortcut | Action |
|----------|--------|
| `Enter` | Save new recipe title |
| `Esc` | Cancel and close dialog |

## Command Palette (`⌘K`)

The command palette is your fastest way to access recipes and actions.

<Steps>

### Open the palette

Press `⌘K` (Mac) or `Ctrl+K` (Windows/Linux) from anywhere in Mizen.

### Type to search

Start typing a recipe name or action. Results filter as you type.

### Navigate results

Use arrow keys to move up/down through results.

### Select

Press `Enter` to open the selected recipe or execute the selected action.

### Close

Press `Esc` or click outside the palette to close without selecting.

</Steps>

## Sidebar Shortcuts

Shortcuts for sidebar navigation:

| Action | Method |
|--------|--------|
| Toggle Sidebar | Click collapse/expand icon (desktop only) |
| Reorder Recipes | Click and drag grip handle (⋮⋮) |
| Quick Actions | Right-click recipe (or long-press on mobile) |

## Settings Menu Search

In the Cook tab settings menu (three-dot icon):

<Steps>

### Open settings menu

Click the three-dot icon in the Cook tab header.

### Search settings

Type in the search box to filter available options (font, step sizing, actions).

### Navigate

Filtered results update instantly as you type.

</Steps>

## Accessibility Notes

### Focus Indicators

All interactive elements show focus indicators when navigating with `Tab`. This helps you see where you are when using keyboard navigation.

### Skip Links

Use `Tab` to cycle through interactive elements in order:
1. Navigation items
2. Recipe cards/buttons
3. Form inputs
4. Action buttons

### Screen Readers

Mizen includes ARIA labels and semantic HTML for screen reader compatibility. All shortcuts and interactive elements announce their purpose.

## Platform Differences

### macOS

Use the `⌘` (Command) key for all shortcuts listed as `⌘`.

### Windows & Linux

Use the `Ctrl` key where `⌘` is shown.

The shortcuts are detected automatically based on your platform.

## Tips for Efficiency

<Tip>
**Learn Tab Shortcuts First**: The tab shortcuts (`⌘1`, `⌘2`, `⌘3`) are the most useful for cooking. Learn these first to navigate effortlessly while cooking.
</Tip>

<Tip>
**Use Command Palette for Recipe Search**: Instead of scrolling through the sidebar, press `⌘K` and type a few letters of the recipe name for instant access.
</Tip>

<Tip>
**Toggle Focus View While Cooking**: Press `⌘J` to switch to Focus view when you start cooking, then use it again to return to List view when you need to see multiple steps.
</Tip>

<Tip>
**Combine with Mouse**: You don't have to use only keyboard or only mouse. Combine both for maximum efficiency—use shortcuts for navigation and mouse/touch for checking off items.
</Tip>

## Customization

Currently, keyboard shortcuts are not customizable. The defaults are designed to:
- Work consistently across platforms
- Avoid conflicts with browser shortcuts
- Follow common application conventions

If you have suggestions for new shortcuts, use the feedback dialog in the app.
