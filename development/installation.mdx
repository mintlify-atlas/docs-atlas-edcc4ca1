## Prerequisites

Before you begin, ensure you have the following installed:

- **Node.js**: Version 24.4.1 (specified in `.nvmrc`)
- **npm**, **yarn**, or **pnpm** package manager
- **Git** for version control

## Node.js Version Management

Mizen requires Node.js version 24.4.1. We recommend using [nvm](https://github.com/nvm-sh/nvm) (Node Version Manager) to manage Node.js versions.

### Installing the Correct Node.js Version

<Steps>
  <Step title="Install nvm">
    If you don't have nvm installed, follow the [installation instructions](https://github.com/nvm-sh/nvm#installing-and-updating).
  </Step>

  <Step title="Install and use the correct Node.js version">
    The project includes a `.nvmrc` file that specifies the required Node.js version:

    ```bash
    # Install the version specified in .nvmrc
    nvm install

    # Switch to the correct version
    nvm use
    ```
  </Step>

  <Step title="Verify installation">
    ```bash
    node --version
    # Should output: v24.4.1
    ```
  </Step>
</Steps>

## Installing Dependencies

Once you have the correct Node.js version, install the project dependencies.

<Steps>
  <Step title="Clone the repository">
    ```bash
    git clone <repository-url>
    cd mizen
    ```
  </Step>

  <Step title="Install dependencies">
    <CodeGroup>
      ```bash npm
      npm install
      ```

      ```bash yarn
      yarn install
      ```

      ```bash pnpm
      pnpm install
      ```
    </CodeGroup>
  </Step>
</Steps>

## Key Dependencies

Mizen is built with the following core technologies:

### Framework & Core

- **Next.js** (v16.1.1) - React framework with App Router
- **React** (v19.2.3) - UI library
- **TypeScript** (v5) - Type-safe JavaScript

### AI & Data Processing

- **Groq SDK** (v0.25.0) - AI-powered recipe parsing
- **Anthropic SDK** (v0.71.2) - Additional AI capabilities
- **OpenAI** (v5.6.0) - OpenAI API integration
- **Cheerio** (v1.1.0) - HTML parsing and scraping

### UI Components

- **Radix UI** - Accessible component primitives
- **Tailwind CSS** (v4) - Utility-first styling
- **Framer Motion** (v12.23.25) - Animations
- **Lucide React** (v0.519.0) - Icon library

### Backend & Database

- **Supabase** (v2.93.3) - Database and authentication
- **Axios** (v1.10.0) - HTTP client

### Development Tools

- **ESLint** (v9) - Code linting
- **Prettier** (v3.6.2) - Code formatting
- **Husky** (v9.1.7) - Git hooks

## Development Scripts

After installation, you can use these npm scripts:

<CodeGroup>
  ```bash Development Server
  npm run dev
  ```

  ```bash Production Build
  npm run build
  ```

  ```bash Start Production Server
  npm start
  ```

  ```bash Lint Code
  npm run lint
  ```

  ```bash Fix Lint Issues
  npm run lint:fix
  ```
</CodeGroup>

## Verify Installation

To verify your installation is working correctly:

<Steps>
  <Step title="Start the development server">
    ```bash
    npm run dev
    ```
  </Step>

  <Step title="Open your browser">
    Navigate to [http://localhost:3000](http://localhost:3000)
  </Step>

  <Step title="Check the console">
    Ensure there are no error messages in the terminal or browser console.
  </Step>
</Steps>

<Note>
If you encounter any issues during installation, check that:
- You're using Node.js version 24.4.1
- All environment variables are properly configured (see [Environment Setup](/development/environment-setup))
- Your package manager is up to date
</Note>
